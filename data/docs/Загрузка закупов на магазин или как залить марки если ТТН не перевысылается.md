Загрузка закупов марками, чтобы они могли делать возвраты 

Тоже после обрезки 
И им надо сделать возврат 
Но у них не выгружается ТТН 

1. Зайти в магазин по марке найти ТТН
2. В буферке открыть Егаис - приём закупа, ввести магазин 
3. Найти эту ТТН(поставить курсор на столбец номер ТТН, затем Ctrl + F), открыть её, затем действие - на основании - ЕГАИС ТТН отправленных
4. Сохранить в XML, назвать номер магазина + номер ТТН (через точку)
5. Удалить все в ТТН в первой вкладке и последней (марки)
6. Затем во вкладке ЕГАИС - обработки закупа (плюсик) пометить этот файл на удаление 
7. В магазине (обязательно в тени): операции - обработки - загрузка марок из ТТН
8. Загрузить этот XML файл с марками в магазин
